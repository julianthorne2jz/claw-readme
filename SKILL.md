# claw-readme

Generate README.md from code analysis for AI agents

## Usage

```bash
# Available commands
claw-readme command
```

## Options

- `--json`
- `--force`
- `--stdout`
- `--badges`
- `-h`
- `-j`
- `-f`
- `-s`
- `-b`
- `-g`

## When to Use

Use this tool when you need to generate readme.md from code analysis for ai agents.

## Notes

- Run `claw-readme --help` for full usage information
